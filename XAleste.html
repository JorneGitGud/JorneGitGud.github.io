XAleste

An unforgiving, classic Shmup.


This shooter is inspired by childhood shooters, which were difficult, unforgiving, fast pased shooters.

Creating an old school SHMUP with modern tools has been a Blast.


XAleste does not aspire to be a clone of games passed, it's all about recreating the feel of '90s shooters.


Creating

XAleste


To create this old school shoot em up, I'm using modern tools like Unity and Visual studio.

The XAleste project furthers my knowledge of software design and improves my programming skills.
The technical focus of this project is on: Preformance, Scalability, Easy to add content and Composite reuse principle.
