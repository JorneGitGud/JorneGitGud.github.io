XAleste

An unforgiving, classic Shmup.


This shooter is inspired by childhood shooters, which were difficult, unforgiving, fast pased shooters.

Creating an old school SHMUP with modern tools has been a Blast.


XAleste does not aspire to be a clone of games passed, it's all about recreating the feel of '90s shooters.

Update


The latest update contains a Upgrade and Pause menu.

Also... Since it had been snowing here in the Netherlands inspiration struck and developement on scene 3-1 had started.

Update

December


This update to XAleste brings the Life system, the player can retry the current level until all lives are lost.

Boosting, turning the player invisible for a short period. Some enemies can be destroyed while boosting

Also; the game can now be paused, better preformance, scaleable with different screen size.
